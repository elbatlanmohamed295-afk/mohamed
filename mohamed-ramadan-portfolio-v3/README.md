- افتح ملف `index.html` في أي متصفح حديث

### 2. الوصول إلى لوحة التحكم
- افتح ملف `admin/index.html`
- بيانات الدخول الافتراضية:
  - **اسم المستخدم:** `admin`
  - **كلمة المرور:** `123456`

### 3. تغيير الصور
- ضع صورتك في `images/profile.png`
- ضع شعارك في `images/logo.png`

## 📝 كيفية التعديل

### تغيير النصوص
افتح `index.html` وابحث عن `data-ar` و `data-en`

### تغيير الألوان
افتح `css/style.css` وعدل متغيرات CSS في `:root`:
```css
:root {
    --primary-color: #0044cc;    ← اللون الأساسي
    --secondary-color: #0a0a1a;  ← اللون الثانوي
    --accent-color: #00d4ff;      ← لون التمييز
}
```

### إضافة مشروع جديد
من لوحة التحكم → إدارة الأعمال → إضافة عمل جديد

## ⚠️ ملاحظات مهمة

1. **البيانات** تُحفظ في `localStorage` المتصفح
2. **الصور المرفوعة** تُحفظ كـ Base64 (حجمها محدود)
3. **للإنتاج الحقيقي** يُنصح بربط الموقع بـ Backend + قاعدة بيانات
4. **غيّر الباسورد** في `admin/js/admin.js`
